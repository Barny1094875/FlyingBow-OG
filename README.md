# FlyingBow-OG
Adds a flying bow to Minecraft

When the bow is shot, the player rides along with the arrow

Current Target: Purpur 1.18.2

Authors: Barny1094875.

<h3>To build:</h3>

./gradlew build

The resulting .jar file will be in build/libs
